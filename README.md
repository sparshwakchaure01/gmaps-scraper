# Google Maps Lead Scraper
## Read the readme file in the zipped folder for setup


A production-ready desktop application that automates Google Maps lead generation for agencies, freelancers, and businesses.

Simply enter a search keyword (e.g., **"Cafe Sangamner"**, **"Footwear Shop Pune"**, **"CA Nashik"**), choose the maximum number of results, and the application automatically collects business information from Google Maps and exports it to a professional Excel (.xlsx) or CSV file.

## Features

* Automated Google Maps search
* Opens every business individually
* Human-like browser automation using Playwright
* Extracts:

  * Business Name
  * Category
  * Phone Number
  * Address
  * Website
  * Website Type (Website / Instagram / WhatsApp / None)
  * Website Status (Working / Broken / Social Only)
  * Rating
  * Review Count
  * Google Maps Link
* Smart lead prioritisation
* Excel (.xlsx) export with formatting
* CSV export
* Live scraping progress
* Resume-friendly error handling
* Duplicate detection
* Configurable maximum results
* No API keys required

## Workflow

1. Enter a Google Maps search keyword.
2. Select the maximum number of businesses.
3. Click **Start Scraping**.
4. The application automatically opens each business and extracts available information.
5. Download the generated Excel or CSV file.

This project is designed to automate repetitive lead collection while keeping the workflow simple, fast, and export-ready for CRM systems or outreach campaigns.
